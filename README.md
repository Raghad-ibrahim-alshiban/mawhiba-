# Mawhiba Information Portal 🌟

Welcome to my very first web development project! This project holds a special place in my journey as a Computer Science student, as it was the first time I combined design and logic to build a complete website.

## 📖 Project Overview
This website is a dedicated platform for Mawhiba (King Abdulaziz & his Companions Foundation for Giftedness and Creativity). The goal was to create a user-friendly interface that introduces the foundation to the public.

### What’s Inside?
- History & Origin: How Mawhiba started.
- Vision & Goals: What the foundation aims to achieve for talented students.
- Foundation Programs: Information about the various initiatives offered.

## 🛠️ Skills & Technologies
In this project, I applied my early knowledge in:
- HTML5: Crafting the structure and content.
- CSS3: Designing the layout and visual elements (UI).
- JavaScript: Adding basic interactivity to the site.

## 🎓 Learning Journey
This project was developed for the "Web Applications" course. It taught me the fundamentals of web architecture and the importance of User Experience (UX) when presenting informational content.
